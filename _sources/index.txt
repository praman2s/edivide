eDiviDe
===================================

This page is a sub part of eDiViDe project. 
The behaviour of the robot is implemented in 'C' which is expected to be translated into VHDL.

The behaviours planned to be implemented are as follows :

* Dead Reckoning.
* Localization of Robot with help of markers.
* Frame assignment in the arena.
* Navigation Control for the robot.
* Robot Exploration.
* Planning Strategy for collection of nuggets.
